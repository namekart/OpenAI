# Site Vision: OpenBusiness.ai
**Autonomous AI Orchestration for Modern Business**

OpenBusiness is a unified platform that adds an "AI Brain" to existing CRM and ERP systems. The goal is to maximize efficiency through voice agents, intelligent automation, and predictive insights.

## 1. Project Context
- **Base Stack:** React + Vite + Tailwind CSS V4 + Framer Motion.
- **Stitch Project ID:** 7965057123801997928

## 2. Core Messaging
- "Add an AI Brain to your existing systems."
- "No migration, no disruption."
- "Voice-first, intelligence-layered."

## 3. Completed Pages (Sitemap)
- [x] **Lead Prioritization Screen** (`/prioritization.html`): AI-driven dashboard for hot leads.
- [x] **Pricing Page** (`/pricing.html`): Comprehensive standalone pricing and FAQ.
- [ ] **AI Deal Risk Detection** (`/crm/risk`): Visualization of deal health and warning signs.
3. **Meeting Summaries & Notes** (`/crm/summaries`): Interface for viewing AI-generated meeting insights.
4. **Smart Data Import** (`/crm/import`): An intuitive UI for mapping CSVs using AI.
5. **Sales Performance Insights** (`/crm/insights`): High-level trends and cross-deal patterns.

## 5. Creative Freedom / Ideas
- **Platform Analytics:** A unified view across both CRM and ERP.
- **AI Agent Configuration:** A space for users to "coach" their AI voice agents.
- **Integration Marketplace:** Showing the breadth of secondary systems we connect to.
- **Case Studies / ROI Calculator:** Interactive tool for leads to see potential savings.
